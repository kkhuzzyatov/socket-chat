# NIO Chat

Простой чат на Java с использованием **NIO** и логирования через **Log4j 2**.

## Особенности

* Поддержка нескольких клиентов одновременно.
* Отправка и получение сообщений в реальном времени.
* Ограничение: имя ≤ 50 символов, сообщение ≤ 200 символов.
* Логирование всех событий (подключение, сообщения, ошибки).

## Как запустить

### Сервер

1. Скомпилировать:

```bash
javac -cp log4j-core.jar;log4j-api.jar ru/ChatServer.java
```

2. Запустить:

```bash
java -cp .;log4j-core.jar;log4j-api.jar ru.ChatServer
```

3. Ввести порт (например, 8080).

### Клиент

1. Скомпилировать:

```bash
javac -cp log4j-core.jar;log4j-api.jar ru/ChatClient.java
```

2. Запустить:

```bash
java -cp .;log4j-core.jar;log4j-api.jar ru.ChatClient
```

3. Ввести адрес сервера, порт и имя пользователя.
